# Momentum -  Task Manager

Momentum is a feature-rich task management application designed for productivity. It helps you organize your tasks with categories, priorities, and due dates, all saved locally in your browser.

*Innovation Lab Project - JS LAB*

![Momentum Screenshot](MOMENTUM.png)

## üöÄ Features

* **Task Management:** Add, edit, and delete tasks.
* **Task Status:** Mark tasks as complete or pending.
* **Task Details:** Assign priorities (High, Medium, Low) and due dates.
* **Custom Categories:** Create and delete custom-colored categories (e.g., "Work", "Personal") to organize tasks.
* **Dynamic Filtering:** Quickly filter your task list by:
    * All Tasks
    * Pending
    * Completed
    * High Priority
    * Due Today
    * Due This Week
* **Statistics:** At-a-glance stats for Total, Completed, and Pending tasks.
* **Interactive Calendar:** A full-month calendar that highlights days with pending tasks. Click any day to filter tasks for that date.
* **Upcoming Tasks:** A dedicated widget showing tasks due in the next 7 days.
* **Data Persistence:** All tasks and categories are saved in the browser's `localStorage`.
* **Responsive Design:** The layout adapts to desktop, tablet, and mobile screens.
